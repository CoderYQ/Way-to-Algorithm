Bubble Sort

冒泡排序

outline: \[m=y\] 问题：
  用冒泡排序对长度为\(n\)的无序序列\(s\)进行排序。

解法：
  本问题对无序序列\(s\)升序排序，排序后\(s\)是从小到大的。
  将长度为\(n\)的序列\(s\)分为\(left\)和\(right\)两个部分，其中\(left\)是无序部分，范围为\(s[0,k])，\(right\)是有序部分，范围为\(s[k+1,n-1]\)，其中\(0\leqk\len\)。初始时\(left\)范围为\(s[0,n-1]\)，\(right\)为空。
