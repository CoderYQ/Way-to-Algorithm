# Way to Algorithm - 算法之路

--------

### Introduction - 简介

本书围绕大学生计算机算法教程与源码，收集和整理了比较常见的计算机算法与数据结构。

其中借鉴了一些经典的算法书籍，和互联网上的资料，结合自己的理解写成。

由于个人经验有限，难免存在一些疏漏和错误的地方，还请斧正。
